Working With Projects

Squinter works primarily through project files. These record which agent and device code files comprise a given model, and which libraries they use. Project files are identified by the .squirrelproj extension. Squinter lets you create new projects, either from scratch or from existing agent and device code files. You can also create projects from your existing models stored in the Electric Imp Cloud.

Projects are listed in the ‘Currently Open Projects’ submenu in the ‘Project’ menu, which also presents actions you can perform with the currently selected project. You can also select a project from the ‘Current Project’ popup menu in Squinter’s main window.

New projects are saved in Squinter’s working directory, which you can set in the Preferences.
