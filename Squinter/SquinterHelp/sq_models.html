Working With Models

Squinter lists all of the models registered to your Electric Imp account, and you can bind these to existing projects or use them as the basis for new ones. You can delete or rename your models, or by creating a new project, create a new model too. You’ll find these options in the ‘Models’ menu.

Selecting a project will automatically select the model it has been bound to, if it has such a link. You can connect a model to the currently selected project using actions in the ‘Project’ and ‘Models’ menus.

You can assign or re-assign a device to any model at any time using a ‘Models’ menu option.
